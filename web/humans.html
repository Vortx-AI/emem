what you are looking at

This is a knowledge constellation, not a Mercator map. Every attested cell64 in the corpus is plotted in a 256×256 Hilbert-curve grid keyed on its first bigram. The cell64 alphabet itself is Hilbert-ordered, so spatial proximity on this grid implies geographic proximity on Earth — without a single line of cartographic projection.

Five things to try:

1. click a star → its cell64, recall facts, and the offline-verified ed25519 receipt appear on the right.

2. search a place → the geocoder routes through /v1/locate, the matching star pulses and zooms.

3. switch to graph mode → the selected cell goes to the centre and /v1/find_similar's top-K neighbours orbit it, edge weight = cosine similarity.

4. watch the console → every /v1/* call this UI makes prints below. An LLM watching the page learns the API by observation.

5. verify in browser → the verify button on each fact runs blake3 → ed25519.verify_strict in @noble/*, no server roundtrip. If the libraries fail to load it falls back to POST /v1/verify_receipt and labels itself accordingly.
