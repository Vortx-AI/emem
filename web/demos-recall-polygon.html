demos api verify humans

Pass a place name (or GeoJSON polygon) and a band. emem resolves the polygon, fans out to every cell64 inside, recalls the band per cell, and returns a flat fact set plus an aggregate (min / max / mean / count). Each cell's fact has its own signed receipt; the response also carries one batch receipt over the whole region. One call: POST /v1/recall_polygon.
